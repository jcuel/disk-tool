What disk-tool does

What's eating my disk?

Overview % breakdown, then drill into heavy folders on demand — no full-tree walk upfront.

Leftover dev junk

Detects node_modules, .venv, target/, and common caches.

Old installers

Flags large .exe, .zip, .msi files in Downloads and similar folders.

Support tickets

Copy a report or export a plain-text support ticket for IT — treemap, tree, and insights included.

Safe cleanup zones

disk-tool classifies paths so you know what is safe to review or delete.

ZoneMeaningDelete via disk-tool?
reviewRegenerable dev artifactsYes, after review
maintenanceUser temp foldersYes, with age filter
cautionShared caches (.npm, .gradle)Review carefully
diagnosticCrash dumps, minidumpsView only
critical_osWindows, /usr, Program FilesNo
forbiddenSystem32, /procHidden from scan

In the app

Quick start

Requires Node.js and Go. See the repo for Docker and CI details.

cd web && npm ci && npm run build && cd ..
cp -r web/dist/* cmd/disk-tool/static/
go build -o bin/disk-tool ./cmd/disk-tool
./bin/disk-tool serve
