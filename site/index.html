What disk-tool does

What's eating my disk?

Overview % breakdown, then drill into heavy folders on demand — no full-tree walk upfront.

Leftover dev junk

Detects node_modules, .venv, target/, and common caches.

Old installers

Flags large .exe, .zip, .msi files in Downloads and similar folders.

Docker reclaim

Maintenance preset runs a confirmed docker system prune (volumes kept). Needs Docker CLI + daemon — never deletes Desktop data folders or VHDX files.

Support tickets

Copy a report or export a plain-text support ticket for IT — treemap, tree, and insights included.

Safe cleanup zones

disk-tool classifies paths so you know what is safe to review or delete. Deletes use review → dry-run → type DELETE. Virtual disks (.vhd/.vhdx/…) stay non-deletable.

ZoneMeaningDelete via disk-tool?
reviewRegenerable dev artifactsYes, after review
maintenanceUser temp foldersYes, with age filter
cautionShared caches, Docker data rootsReview carefully (Docker roots not FS-deletable)
diagnosticCrash dumps, minidumpsView only
critical_osWindows, /usr, Program FilesNo
forbiddenSystem32, /procHidden from scan

In the app

Quick start

Download a pre-built binary from Releases (Windows, Linux, macOS), or build from source below. 

# After download
./disk-tool serve

# Or build from source (Node.js + Go)
cd web && npm ci && npm run build && cd ..
cp -r web/dist/* cmd/disk-tool/static/
go build -o bin/disk-tool ./cmd/disk-tool
./bin/disk-tool serve
