# User story: GitHub Pages site

As a **prospective disk-tool user**, I want a public site that explains the product and lets me try a sample scan in the browser, so I can evaluate disk-tool before installing it locally.

## Acceptance criteria

- [ ] Landing page at `/disk-tool/` with hero, features, safety zones, screenshots, install CTA
- [ ] Interactive demo at `/disk-tool/demo/` using the real UI with sample data
- [ ] Demo shows a banner that actions like delete/open require local install
- [ ] Site deploys automatically on push to `master`
- [ ] PR CI verifies site and demo builds
